# Misc Agent Configs

This folder holds all non-primary config profiles.

The top-level `agent_configs/` directory is intentionally reserved for the four
prominent, fully standalone E4 reference dossiers:

- `codex_0-107-0_e4_3-6-2026.yaml`
- `claude_code_2-1-63_e4_3-6-2026.yaml`
- `opencode_1-2-17_e4_3-6-2026.yaml`
- `oh_my_opencode_3-10-0_e4_3-6-2026.yaml`

Everything else lives here so the repo surface stays legible while preserving
historical overlays, experiments, smoke configs, and narrower replay lanes.
