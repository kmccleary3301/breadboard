# Validation gap matrix

This census is grounded in the live `p20/wsI` tree. It contains 25 agent-config loader checks and 8 helper-runtime checks, matching `SCOUT_FACTS.json` S3. `schema_covered` describes the current `bb.agent_config_surface.v2` schema only; helper compiler inputs are outside that schema. A schema pointer in a `delete` decision identifies the keyword that already enforces the check. A `schema-first` decision identifies the schema section that I2 must tighten before I3 removes the Python check.
