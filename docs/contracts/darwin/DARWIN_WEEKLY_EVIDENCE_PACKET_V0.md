# DARWIN Weekly Evidence Packet V0

The weekly packet is the default operational review artifact for Phase-1.

## Required contents

- run manifests emitted this week
- lane scorecard references
- budget burn summary
- drift and incident summary
- contamination / exclusion summary
- promotion decisions
- rollback decisions
- next-week actions

The machine-readable packet contract is `schemas/weekly_evidence_packet_v0.schema.json`.
