# DARWIN Stage-4 Component Family Policy V0

Date: 2026-03-20
Status: active for the Stage-4 multi-family promotion tranche

## Purpose

Define the only promotable Stage-4 family types for the first compounding tranche.

## In scope

- topology families
- tool-scope families
- policy families

## Out of scope

- arbitrary prompt fragments
- route artifacts
- untyped arm bundles
- lane-local hacks presented as reusable families
- broad memory-state artifacts

## Family versus run artifact

A run artifact is one concrete bounded live or replayed outcome.

A family is a typed reusable unit inferred from repeated valid live comparisons under the Stage-4 matched-budget and replay rules.

## Promotion-eligibility minimum

A Stage-4 family is promotion-eligible only when:

- repeated valid live comparisons exist
- provider-backed claim eligibility exists on the supporting live rows
- matched-budget validity holds
- replay posture is recorded
- prior stability is not chaotic

## Tranche note

This tranche exists to establish the first canonical Stage-4 family program. It does not authorize broad family discovery, broad transfer search, or archive-as-truth behavior.
